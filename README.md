# chapter9-RemoteServer

## NodeRED instance
Code application for the instance of NodeRED to run in the remote server (Middleware Layer)

How to start the application:
```
$ cd ./nodered
$ bash start.sh
```

## InfluxDB UI
Web interface for accessing the InfluxDB database
```
$ cd ./influxdb-ui
$ npm install
$ npm run build
$ npm start
```
